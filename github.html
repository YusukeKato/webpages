リポジトリ作成 -> githubの案内に従って、初回のpush
git add file_name （変更全部の場合： [--all]　便利 ）
git commit -m "コメント" （ -m は、コメントのオプション）
git push origin branch_name （ origin は、デフォルトのリポジトリ名(URL?) ）
==================================================
add       - リポジトリ変更情報をインデックスへ ステージング
commit    - リポジトリのインデックスに追加された変更をコメントをつけてコミット
push      - ローカルのコミットをリモートのリポジトリに反映
pull      - リモートからローカル
branch    - ブランチ表示　引数にブランチ名でブランチ作成
checkout  - 引数にブランチ名で、そのブランチに移動
merge     - 引数にブランチ名で、そのブランチの内容を取り込む
            （ローカルとリモートで同じ個所を書き換えていると、コンフリクト）
