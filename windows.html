＜コマンドプロンプト＞

cd    - カレントディレクトリの変更(chdir)
copy  - ファイルのコピー
xcopy - ディレクトリのコピー
del   - ファイル削除(erase)
dir   - ディレクトリの情報表示
md    - ディレクトリ作成(mkdir)
move  - ファイルの移動、ディレクトリ名の変更
rd    - ディレクトリ削除(rmdir)
ren   - ファイル名の変更(rename)
replase - ファイルの置き換え
tree  - ディレクトリのツリー表示

cls   - 画面クリア
exit  - 終了
start - 開始
