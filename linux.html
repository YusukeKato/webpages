      ==================================================
      cat   - ファイルの中身を表示、複数のファイルの連結
      cd    - カレントディレクトリの変更（今見てるディレクトリ）
      cp    - コピー、cp [option][[コピーするファイル名][コピー先]
      clear - 画面クリア
      kill  - プロセス、ジョブを終了させる
      ls    - ディレクトリの中を見れる
      man   - コマンドのマニュアル表示
      mkdir - ディレクトリ作成
      mv    - ファイルの移動、または、名前変更
      ps    - プロセスを表示
      pwd   - カレントディレクトリの表示
      rm    - ファイル、ディレクトリの削除（危険）
      touch -
      vi    - エディタ
      ed    - エディタ（軽い）
      ==================================================
      ＜オプション＞（少しだけ）
      cat -n  = 行番号を付けて表示
      ls -l   = 詳しいデータも表示
      cp -i   = 上書きするか確認
      mv -i   = 同上
      rm -r   = ディレクトリを削除
      sort -r = 逆順にソート
      ==================================================
      ＜パイプ＞
      コマンドを[|]でつなげて書く
      前のコマンドの出力を、次のコマンドの入力に渡せる
      ==================================================
      cd ..   - 一つ上のディレクトリに変更

      grep 検索
